# cadcad-rust-poc
Proof of Concept Rust Implementation of cadCAD

## How to experiment 
git clone repo  
`cd cadcad-rust-poc`  
`cargo r`

#### C. HashMap vs BTreeMap perf. test - with config_prey_predator_integer.rs (5-Dec-21):

**Summary:**
