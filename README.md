# cadcad-rust-poc
Proof of Concept Rust Implementation of cadCAD

## How to experiment 
git clone repo  
`cd cadcad-rust-poc`  
`cargo r`

### Performance

#### HashMap vs BTreeMap test - with config_prey_predator_integer.rs (ver. 5-Dec-21):

Summary:
